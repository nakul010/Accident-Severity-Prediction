# Accident-Severity-Prediction 

## Getting started

1. Clone the repository:

   ```text
   git cone https://github.com/nakul010/Accident-Severity-Prediction
   ```

2. Install the required dependencies:

   ```text
   pip install -r requirements.txt
   ```
   
3. Run the app.py
   ```text
   streamlit run app.py
   ```
